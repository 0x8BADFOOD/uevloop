Detailed Description

The scheduler object.

This object keeps track of time run since the application was launched. It also keeps a queue of events due to be processed sometime in the future.

cqueue_t* scheduler_t::event_queue

The system's event queue.

When an event's due time arrive, they will put in this queue for further collection and processing by the event loop.

◆ reschedule_queue

cqueue_t* scheduler_t::reschedule_queue

The system's reschedule queue.

Events put in this queue will be reschedule according to their due time by the next time the scheduler manages its timers.

◆ timer_list

llist_t scheduler_t::timer_list

Scheduled timers linked list.

This linked list holds events/timers scheduled to be run in the future. Timers are inserted sorted by its due time, so it is always in execution order.

Expired timers may be returned here if they are repeating.
